components/TodoItem/_TodoItem.js: one card where model() shows its magic

This is the best "look, it is really used" slide. One todo card has checkboxes, text inputs, selects, a textarea, and a delete button, but they all stay tiny because the helpers do the boring work.
  • Checkboxes use prop: "checked" and event: "change" so DOM booleans map cleanly to store booleans
  • Title, notes, category, priority, and due date all reuse the same todoModel() helper
  • The delete button is just @click, so event parts and model parts live together happily
  • The card also mirrors state into data-state and data-priority, which keeps styling honest and declarative
