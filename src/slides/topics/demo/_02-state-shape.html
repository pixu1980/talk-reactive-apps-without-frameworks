Persistent state

  • todos array with title, notes, category, priority, due date, completion and selection flags
  • categories list reused by draft form and todo cards
  • draft values for quick add
  • filters for search, category, status, priority and sorting

Ephemeral-but-visible state

  • debug.paused toggles the event panel
  • debug.logs stores the latest 30 mutation entries
  • mainState is the render invalidation bridge
  • summary, visibleTodos and debugLogs are computed selectors over the store
