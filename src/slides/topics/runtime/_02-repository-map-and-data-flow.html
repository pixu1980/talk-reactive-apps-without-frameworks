Repository map

  • reactive-apps-without-frameworks-demo/src/scripts/core/store/* Proxy tree, snapshot cloning, store:change emission
  • reactive-apps-without-frameworks-demo/src/scripts/core/signals/* State, Computed, dependency tracking, microtask effects
  • reactive-apps-without-frameworks-demo/src/scripts/core/template-engine/* template compilation, parts, directives, keyed repeat
  • reactive-apps-without-frameworks-demo/src/scripts/state/* store setup, computed selectors, actions, persistence bridge
  • reactive-apps-without-frameworks-demo/src/scripts/views/app/* app shell subcomponents, panels and debug surface
  • reactive-apps-without-frameworks-demo/src/scripts/data/* seed state, generator filters, sorting pipeline

Data flow

  • A user action calls a store mutation or an action function
  • The store emits a store:change event with path, oldValue, newValue
  • The state bridge persists the snapshot, appends a debug entry, then bumps mainState
  • Signals and template parts update the minimum DOM needed for the new view state
